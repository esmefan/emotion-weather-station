# 情绪气候站 (Emotion Weather Station)

一个将情绪视为天气系统的心理健康移动端Web应用。

## 功能特点

### 1. 情绪输入界面
- 四种主要情绪状态：焦虑、空虚、烦躁、倦怠
- 大尺寸触控友好的情绪按钮
- 可选自由文本输入记录详细感受

### 2. 天气系统生成器
将情绪映射为天气隐喻：
- **焦虑** → 🌩️ 雷暴焦虑 (深蓝/紫色调)
- **空虚** → 🌫️ 低压空虚 (灰/雾色调)
- **烦躁** → 🌬️ 失控风 (琥珀/橙色调)
- **倦怠** → 🌧️ 持续阴雨 (石板/青色调)

每个天气卡片包含：
- 情绪天气标题和诗意描述
- 气候强度计 (1-5级)
- 个性化情绪预报提示

### 3. 干预面板 ("气候干预方案")
- **🎵 声音推荐**：三种环境音选项 (雨声/电台白噪音/低语人声)
- **🚶 微行为**：温和的行为提示，可随机刷新
- **🌊 呼吸引导**：动画呼吸波浪，4秒吸气/呼气循环

### 4. 导航系统
- **今日**：情绪输入和干预主界面
- **气候日志**：28天情绪日历，可视化情绪历史
- **设置**：应用偏好和数据管理

## 设计理念

- **移动优先**：专为移动设备优化的响应式设计
- **深色模式**：柔和暗色主题，减少视觉疲劳
- **天气隐喻**：将抽象情绪具象化为可感知的天气系统
- **微干预**：小而可行的情绪调节建议
- **数据隐私**：所有数据仅存储在本地设备

## 技术栈

- **前端**：纯HTML5、CSS3、JavaScript (无框架依赖)
- **样式**：自定义CSS变量系统，Tailwind风格实用类
- **图标**：Font Awesome 6 + 原生Emoji
- **字体**：Inter + Noto Sans SC (中文字体)
- **存储**：浏览器LocalStorage API

## 快速开始

1. 克隆或下载项目文件
2. 直接在浏览器中打开 `index.html`
3. 或在本地运行HTTP服务器：
   ```bash
   # Python 3
   python -m http.server 8000
   
   # Node.js
   npx serve .
   ```
4. 在移动设备或浏览器开发者工具中访问

## 使用说明

1. **记录情绪**：点击四种情绪按钮之一
2. **设置强度**：选择1-5级气候强度
3. **查看报告**：浏览个性化情绪天气报告
4. **选择干预**：从声音、行为、呼吸中选择调节方式
5. **跟踪历史**：在日志页面查看情绪模式

## 开发说明

- 所有样式使用CSS变量，便于主题定制
- JavaScript采用模块化设计，状态管理清晰
- 响应式断点：480px (手机)、768px (平板)
- 动画使用CSS关键帧和JavaScript组合

## 设计亮点

1. **情绪可视化**：颜色编码的情绪天气系统
2. **渐进式展示**：情绪选择→天气报告→干预方案
3. **呼吸动画**：SVG波浪实时反映呼吸节奏
4. **历史日历**：28天情绪点阵图，直观模式识别
5. **无障碍设计**：足够的对比度，清晰的视觉层次

## 理念背景

"情绪气候站"基于以下心理学理念：
- 情绪如天气，自然流动且有周期性
- 觉察是情绪调节的第一步
- 微小、可行的行动能改变情绪气候
- 可视化有助于情绪认知和接纳
